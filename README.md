# Introduction

this is a wallet manager script for terminal.

include the following features:

wallet manage:

- create wallet
- import wallet
- export wallet
- list wallet

wallet operation:

- connect to dapps via walletconnect (v1 and v2)
- sign personal_sign message
- sign eth_sign message
- sign transaction
- send transaction

chain manage:

- change active chain

# Usage

```bash

npm install

npm run start

```
